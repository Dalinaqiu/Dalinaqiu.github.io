macos清理空间（进阶版）

Aqiu2026/04/29日志日志

Mac 优化完整提示词 & 操作步骤

写完之前的Mac清理空间教程后，在网上发现了一个更强的教程，记录一下。

原文：https://home.fishbookstudio.cc/blog/3417aed0-d3d3-8005-90f0-dcdc1c80d2fa

⚠️ 开始之前，请务必完成以下准备：

备份你的电脑（Time Machine / iCloud / 移动硬盘，任选其一）

安装 Claude CLI：打开终端，运行 brew install claude 或前往 claude.ai 下载桌面版

打开终端，输入 claude 进入对话模式

第 1 步：全面系统诊断

将以下提示词发送给 Claude 👇

帮我诊断我的 Mac。检查 CPU 使用率、内存、存储空间和正在运行的进程。告诉我哪些在占用资源，以及哪些可以删除。

✅ 你会看到什么： Claude 会运行 top、vm_stat、df -h、ps aux 等命令，输出 CPU 负载、内存使用、磁盘占用和进程列表。

✅ 你需要做什么： 查看 Claude 标记的高占用进程，确认哪些是你不需要的。Claude 会逐项询问你是否要关闭或清理，每一步都需要你确认后才会执行。

第 2 步：清理隐藏缓存和残留文件

查找并删除隐藏缓存、旧的应用支持文件，以及已经卸载应用残留的数据。

✅ 你会看到什么： Claude 会扫描 ~/Library/Caches、~/Library/Application Support、~/Library/Logs 等目录，列出每个缓存文件夹的大小。

✅ 你需要做什么： 逐项确认是否删除。重点关注：

  • 已卸载应用的残留文件夹（如 VisualStudio、Edge、GitKraken）

  • 流媒体临时文件（如 Apple Podcasts 的 StreamedMedia）

  • 开发工具缓存（如 JetBrains、pip、node_modules）

第 3 步：清理开机自启项

检查开机启动和登录时运行的程序。禁用我不需要的内容。

✅ 你会看到什么： Claude 会检查三个位置：

  • 登录项（系统设置 → 通用 → 登录项）

  • LaunchAgents（~/Library/LaunchAgents 和 /Library/LaunchAgents）

  • LaunchDaemons（/Library/LaunchDaemons）

✅ 你需要做什么： Claude 会列出所有自启动项并标注建议操作（禁用/删除/保留）。对于需要管理员权限的操作，你需要输入密码确认。

第 4 步：找出大文件和闲置应用

找出占用存储最多的文件。比如旧的下载文件、重复文件，以及超过 500MB 的闲置应用。

✅ 你会看到什么： Claude 会用 find、du 等命令扫描磁盘，按大小排序列出：

  • Downloads 文件夹中的旧文件和重复压缩包
  • 超过 500MB 且长期未使用的应用
  • 大型媒体文件、虚拟机镜像等

✅ 你需要做什么： 逐项决定是否删除。Claude 会告诉你每个文件最后一次打开的时间，帮你判断是否还需要。

第 5 步：优化系统性能设置

优化我的 Mac 性能设置，比如 GPU 调度、电源偏好设置，以及任何默认处于省电模式的选项。

✅ 你会看到什么： Claude 会检查并调整：

  • 电源管理设置（pmset 命令）
  • GPU 切换模式（自动/高性能）
  • 节能相关的隐藏设置
  • Spotlight 索引状态

✅ 你需要做什么： 根据你的使用场景（插电/移动）选择 Claude 推荐的配置方案。

🎉 完成后：验证优化效果

所有步骤完成后，你可以发送：

再跑一次系统诊断，和之前的数据对比，告诉我优化了多少。 Claude 会生成一份前后对比报告，包括 CPU 负载、内存使用、磁盘空间等关键指标的变化。

💡 注意事项：

每一步操作 Claude 都会先告诉你它要做什么，你确认后它才会执行 需要管理员权限的操作会提示你输入密码 你的系统配置和我不一样，结果会有差异，请根据自己的实际情况判断 如果不确定某个文件能不能删，就先跳过，宁可多留不要误删
