摘要

Aqiu2020/05/29

随着互联网的普及，自媒体和短视频行业蓬勃发展，自由创作者数量飞速增长，并产生了天量的数字作品。与传统作品相比，数字作品无需依赖实物载体，更容易被复制和传播，与此同时，对数字作品的版权保护也愈发困难。

传统的版权保护系统大都采用中心化技术方案，由用户或者组织通过网页、小程序、手机App等入口访问，向中心服务器发起请求。登记周期长、登记流程复杂，并且存在信任问题。在信息爆炸式增长和更新的互联网环境下，将传统的版权保护系统直接应用于数字作品的版权保护并不完全合适。 区块链技术能够及时、有效地记录数字作品版权的产生、使用、交易、许可和转让的过程，解决数字作品版权的确权、交易问题，也可以在技术上对侵权行为进行追踪。同时，其内置共识机制，具有不可篡改性；支持编写智能合约，满足条件的合约可以自动执行。由于这些特性，区块链技术为数字作品版权保护系统提供了一种新的技术选择。

本文首先从数字作品版权保护的需求出发，分析了区块链技术与传统方案相比所具有的技术优势；然后设计数字作品版权保护系统，包含数字作品版权登记、查询、转让、交易等功能，进而基于百度开源的超级链XuperChain完成系统功能的实现。在设计过程中，区块链系统的高并发性、高可用性也是重要考虑因素，该数字作品版权保护系统可以在短时间内处理大量的用户请求，同时也具备一定容错性。最后，总结本文所做工作、创新点以及不足之处。

本文的研究目标是探索实现一个基于区块链技术的数字作品版权保护系统，为个人用户提供一个完整的数字作品版权登记、保护、交易工具。
