研究背景和意义

Aqiu2020/05/28

根据国家互联网信息办公室发布《数字中国发展报告（2022年）》，截至2022年底，我国网民规模达到10.67亿，较去年同期增长3549万人，互联网普及率达75.6%。据今年数据统计，2023年2月移动互联网的用户规模已经开始突破12亿量级，至3月末用户规模已达12.03亿，较去年同期增加2904万。巨大的数据生产、数据消费群体催生出数字作品版权保护、消费的巨大需求。在当前的互联网环境下，数字内容的盗版和侵权问题已经成为一个全球性的挑战。

传统的版权保护系统一般是采用中心化系统设计，用户或者组织通过网页、小程序、手机App等入口访问，向中心服务器发起请求，数据存储方案一般采用MySQL、SQLServer、Oracle等关系型数据库。

图1:中心化系统架构图

传统版权保护系统中心化的系统设计存在如下问题：

首先，传统版权保护系统登记周期长、流程复杂。我国当前版权登记制度采用自愿登记原则，依据我国版权保护中心《作品著作权自愿登记收费标准》的相关规定，作品登记需要一定的费用，且申请者每次登记都需要填写作品登记表、登记机关核查作品、审核资料、发放作品登记证和信息公示等一系列程序，导致申请耗费时间较长。

其次，中心化系统设计存在版权交易失真的问题。个别实力雄厚的网络服务商会与创作者签订独家使用许可的合同，大规模垄断数字作品，一些个人创作者想迅速推广自己的数字作品，必须被动的接受寡头平台提出的低价许可推广合同，这会导致偏离真实的市场价格竞争机制，失去了版权交易许可的真实性。

第三，维权困难。当前我国对版权保护缺乏法律意识和权利意识的公众较多，在自媒体、短视频行业的蓬勃发展的环境下，数字作品极易进行复制和传播，导致版权侵权现象频发，维权困难。

因此，急要寻求创新的解决方案来保护数字作品版权。区块链技术作为一种去中心化、可追溯和不可篡改的分布式账本技术，为数字作品版权保护提供了一种具有潜力的解决方案。当前区块链技术应用步入高速发展阶段，产业生态已初具规模。我国“十四五”规划纲要在2021年3月正式颁布，提出将区块链作为新型数字产业之一，对各领域利用区块链技术促进高质量发展做出了战略部署。

基于区块链技术的数字作品版权保护系统可以通过将版权信息和交易记录存储在区块链上，实现去中心化的版权管理和控制。该技术可以确保版权信息的透明性和不可篡改性，使得数字内容的创建者可以有效地证明其所有权，防止未经授权的复制和传播。

此外，区块链技术还可以支持智能合约功能，可以编写智能合约规定数字内容的使用条件和权益分配机制。这样一来，数字作品版权的管理和控制可以实现自动化和可信赖，减少中间环节以及人为干预的可能性，提高版权保护的效率和准确性。
